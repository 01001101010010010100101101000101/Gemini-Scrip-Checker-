# Gemini-Scrip-Checker-

This repository hosts a simple static site intended for deployment via GitHub Pages.

## Deployment

A GitHub Actions workflow (`.github/workflows/pages.yml`) is configured to automatically publish the contents of the `main` branch to GitHub Pages whenever a push occurs.

- Go to **Settings > Pages** in the repository and verify the source is set to GitHub Actions.
- After the first successful run, the site will be served at:
  `https://01001101010010010100101101000101.github.io/Gemini-Scrip-Checker-/`

You can customize `index.html` or add other assets as needed.
